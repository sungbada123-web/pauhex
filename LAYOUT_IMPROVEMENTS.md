## 📐 具体改进

### 1. 导航栏 (Navigation)
**修改内容：**
- 从 `max-w-7xl` 改为 `max-w-[1400px]`
- 从 `justify-between` 改为 `justify-center` 并添加 `gap-16`
- Logo、菜单、按钮都在居中容器中，间距均匀
- 菜单项间距从 `space-x-8` 增加到 `space-x-10`
- 字体从 `text-sm` 增加到 `text-base`

**效果：**
```
之前: AURAON | Products Solutions...        [Contact]
现在:         AURAON    Products  Solutions  ...    Contact
```

### 2. Hero 区域主标题
**修改内容：**
- 添加 `justify-center` 使整个区域居中
- 添加 `text-center` 使文字居中对齐
- 标题字号从 `text-6xl` 增加到 `text-7xl`
- 副标题从 `text-xl` 增加到 `text-2xl`
- 按钮从 `px-8 py-4` 增加到 `px-10 py-5`
- 添加 `max-w-3xl mx-auto` 限制副标题宽度
- 背景渐变改为均匀覆盖，提高可读性

**效果：**
```
之前:
Wir definieren den...
Basierend auf deutscher...
[Button]

现在:
          Wir definieren den...
    Basierend auf deutscher...
              [Button]
```

### 3. 产品矩阵区域
**修改内容：**
- `max-w-7xl` → `max-w-[1400px]`
- `px-6` → `px-8`
- `py-20` → `py-24`
- 标题从 `text-4xl` → `text-5xl`
- 副标题从 `text-lg` → `text-xl`
- 卡片间距从 `gap-8` → `gap-10`

**效果：**
更宽的内容区域，更大的间距，视觉更舒适

### 4. 权威声明区域

## 📊 布局规范

### 容器宽度标准
```css
导航栏:     max-w-[1400px] + px-8
内容区域:   max-w-[1400px] + px-8
页脚:       max-w-[1400px] + px-8
```

### 间距标准
```css
垂直section间距: py-24
卡片间距:       gap-10
水平padding:    px-8
```

### 字体大小
```css
主标题 (Hero):      text-7xl
区域标题:           text-5xl
副标题:             text-xl - text-2xl
正文:               text-base - text-xl
```
